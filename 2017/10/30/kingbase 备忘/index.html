Kingbase 备忘

发表于 2017-10-30

KingbaseES 是人大金仓的数据库，作为国产数据库在国内项目中越来越被用到。虽然从 Oracle 转到 KingbaseES 金仓做了很大支持，还是有必要记录下 KingbaseES 的使用。

jdbc

  • Class.forName(“com.kingbase.Driver”)
  • url
    • jdbc:kingbase:database
    • jdbc:kingbase://host/database
    • jdbc:kingbase://host:port/database
    • jdbc:kingbase://host:port/database;user=userName;pwd=password;

语句对象

  • Statement 对象：由 createStatement() 创建，发送简单的 sql 语句
  • PreparedStatement 对象：由 prepareStatement() 创建，发送带有输入参数的 sql 语句。
  • CallableStatement 对象：由 prepareCall() 创建，用于执行 sql 存储过程。
